Vaadin TouchKit ${Implementation-Version}

Vaadin TouchKit is an add-on component for Vaadin to aid creating mobile web applications for touch based devices like smartphones and tablets.

Note, that this Vaadin 7 compatible version is still at beta level. We still have some bugs and performance issues to tackle. If you have tight schedule in your project, consider using the Vaadin 6 compatible 2.X branch at this point.

TouchKit 3 (and Vaadin 7) versions of our demo apps are availabels via SVN: Vornitologist and Mobilemail.

Usage of modern build systems like Maven, that supports transient dependencies, is highly encouraged. Things should work fine with following dependency declaration and Vaadin Maven plugin:

		
<dependency>
	<groupId>com.vaadin.addon</groupId>
	<artifactId>vaadin-touchkit-${license.short.name}</artifactId>
	<version>${Implementation-Version}</version>
	<type>jar</type>
</dependency>
