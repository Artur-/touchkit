Vaadin TouchKit ${Implementation-Version}

Vaadin TouchKit is an add-on component for Vaadin to aid creating mobile web applications for touch based devices like smartphones and tablets.

Note, that this Vaadin 7 compatible version is still under active development. Especially documentation and tutorial are not in their final shape. See vornitologist and mobilemail demos from SVN for up to date code examples. We also still have several bugs and performance issues to tackle. If you have tight schedule in your project, consider using the Vaadin 6 compatible 2.X branch at this point.

Usage of modern build systems like Maven, that supports transient dependencies, is highly encouraged. Things should work fine with following dependency declaration and Vaadin Maven plugin:

		
<dependency>
	<groupId>com.vaadin.addon</groupId>
	<artifactId>vaadin-touchkit-${license.short.name}</artifactId>
	<version>${Implementation-Version}</version>
	<type>jar</type>
</dependency>

Dependencies

This version of TouchKit is built against Vaadin ${vaadin.version.maven}, but is expected to be forward compatible with minor versions. In addition to Vaadin, TouchKit depends some third party Apache 2 licensed Java libraries. Currently these libraries are only required during widgetset compilation. Those are automatically resolved for Maven users and also (others but core vaadin jars and their dependencies) are also collected to this packages lib directory.
